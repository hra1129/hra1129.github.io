XEVI避け on SCREEN5 on MSX2
-------------------------------------------------------------------------------

1. はじめに
	動作条件としては、MSX2 VRAM128KB の機種となります。
	MSX1 や MSX2 VRAM64KB, MSX1アダプタによる MSX2 は非対応となりますのでご注意
	ください。

	本プログラムは、MSX2 の SCREEN5 において、アンド○ジェネシスを浮遊させること
	を目的としたデモプログラムです。

2. 流用について
	MSXのための創作への流用は、プログラムの一部または全部を問わず、自由に行って
	いただいて構いません。無償・有償も問いません。

	何かしらの面白いソフトができあがった場合は、提供いただけると嬉しいですが、
	提供を強要するものでもありません。

3. ファイル構成
	MSXで利用するファイル
		AUTOEXEC.BAS ........................ 起動プログラム
		MAP.SC5 ............................. マップデータ(BSAVE形式)
		PAGE1.SC5 ........................... 背景パーツ等の画像データ(SCREEN5, BSAVE形式)
		XEVIYOKE.BIN ........................ 本プログラムのメインルーチン(BSAVE形式)
	その他のファイル
		bgm ................................. BGMデータ
		bgm/compile.bat ..................... xeviyoke.txt をコンパイルして source/bgm.asm を生成するWindows用バッチファイル
		bgm/mc.exe .......................... MMLコンパイラ
		bgm/xeviyoke.txt .................... BGMのMML記述テキスト
		mapedit/xeviyoke_mapeditor.exe ...... XEVI避専用のマップエディッタ
		mapedit/xeviyoke_mapeditor.zip ...... xeviyoke_mapeditor.exe のソースファイル一式(C#)
		source .............................. ソースファイル一式。zma v1.0.5以降でアセンブルできます。
		source/compile.bat .................. *.asm から XEVIYOKE.BIN を作るためのWindows用バッチファイル
		source/*.asm ........................ XEVIYOKE.BIN のソースファイル
		page1.gal ........................... PAGE1.SC5 の ドット絵エディタGraphicGaleデータ(https://graphicsgale.com/)
		page1.bmp ........................... page1.gal を BMP画像として保存したもの
		page1.b2m ........................... page1.bmp を PAGE1.SC5 に変換するための BMP2MSX の設定ファイル
		readme.txt .......................... このファイル
		vdp_bigchar.pdf ..................... 説明資料
		xeviyoke.dsk ........................ MSXで利用するファイルを書き込んだディスクイメージ

4. MSX実機での実行方法
	上記 MSXで利用するファイル4つを MSX-DOS1フォーマットした 2DDディスクに
	書き込んで、そのディスクから起動する。

5. パソコン上のエミュレータ(BlueMSX等)で実行する方法
	xeviyoke.dsk をディスク指定して、そこから起動する。

6. マップエディッタ
	xeviyoke_mapeditor.exe (Windows10用) を起動。
	Load Parts Image ボタンをクリックして page1.bmp を指定するとマップパーツが読み込まれる。
	Load Map ボタンをクリックして MAP.SC5 を指定するとマップが読み込まれる。
	Save Map ボタンをクリックして MAP.SC5 を指定すると編集したマップを保存できる。
	左上のマップパーツの表示領域をクリックすると、赤い四角のカーソルを移動できる。
	右側の領域はマップの表示になっている。右クリックでカーソル位置のパーツを取得、
	左クリックで左上のパーツ表示領域で赤い四角のカーソルの位置のパーツを配置。
	Up ボタンで上に1パーツ分スクロール。
	Down ボタンで下に1パーツ分スクロール。
	Up と Down の間の数値は、マップ上の Y座標。

	スクロールボタンは押しっぱなしが効かないので注意。
	一度クリックしてから、スペースバーを連打するのが楽。

	ここにあまり労力を割きたくなかったので、非常に使い勝手の悪いツールになってます。
