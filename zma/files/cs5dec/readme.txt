-------------------------------------------------------------------------------
画像圧縮伸張
-------------------------------------------------------------------------------

1. はじめに
	動作条件としては、MSX2 VRAM64KB以上 の機種となります。
	MSX1 や MSX2 VRAM64KB, MSX1アダプタによる MSX2 は非対応となりますのでご注意
	ください。

	本プログラムは、MSX2 の SCREEN5 用の画像(BSAVE形式)の全部または一部を圧縮する
	PC用のアプリ cs5codec.exe と、MSX-DOS上で動作する伸張 cs5dec.com のツールに
	なります。
	cs5dec.com のソースファイルを付けていますので、これを利用してゲーム等に組み込
	み、画像の読み込み処理を実施することができます。

2. 流用について
	MSXのための創作への流用は、プログラムの一部または全部を問わず、自由に行って
	いただいて構いません。無償・有償も問いません。

	何かしらの面白いソフトができあがった場合は、提供いただけると嬉しいですが、
	提供を強要するものでもありません。

3. ファイル構成
	MSXで利用するファイル
		CAKE.CS5 .................. サンプル写真(圧縮済み)
		PAGE1.CS5 ................. サンプル画像(圧縮済み)
		CS5DECL.COM ............... MSX-DOS用伸張ツール(LMMC版)
		CS5DECH.COM ............... MSX-DOS用伸張ツール(HMMC版)
		CS5DECV.COM ............... MSX-DOS用伸張ツール(VPOKE版)
	その他のファイル
		for_pc .................... PC用圧縮/伸張ツールのソースファイルと実行ファイル
		src\*.asm ................. CS5DEC*.COM のソースファイル
		readme.txt ................ このファイル
		compile.bat ............... CS5DEC.COM を作るためのWindows用バッチファイル(要zma.exe)
		cs5format.pdf ............. 圧縮ファイルの構造の説明
		old ....................... 素直に作った版のソース差分

4. MSX実機での実行方法
	MSX-DOSの入ったディスクに CAKE.CS5 と CS5DEC.COM をコピーして、
	MSX-DOS上で下記のコマンドを実行。

	A:>CS5DECL CAKE.CS5

	背景色などの変更は行わないため、事前に BASIC で COLOR 15,0,0 しておくと良い。
	CS5DEC* は、SCREEN5 決め打ちでデコードするので、それ以外の画像の場合正常に表示できないので注意。
	CS5DECV は、SCREEN5 のフル画面サイズしかデコードできないので注意。

5. 圧縮ファイルの作り方
	PC上でコマンドプロンプト(cmd.exe)を起動して、cs5codec.exe と、目的の xxx.sc5 が
	置いてあるフォルダへ CD する。

	C:\xxxx> cs5codec xxx.sc5 xxx.cs5

	これで xxx.cs5 ができあがる。
	もし、xxx.sc5 にパレットデータ(COLOR=RESTORE形式)が含まれている場合

	C:\xxxx> cs5codec -palette xxx.sc5 xxx.cs5

	とすることで、圧縮データにパレット情報を含めることができる。
	その他、部分セーブするためのオプションもあるため、詳細は下記のコマンドを打って
	出てくるメッセージを参照。

	C:\xxxx> cs5codec -help
